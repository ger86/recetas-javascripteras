## Similarities

```js
Object.is('Latte and Code', 'Latte and Code'); // true
'Latte and Code' === 'Latte and Code' // true

Object.is({}, {}); // false
{} === {} // false
```

## Differences

```js
Object.is(-0, 0); // false
-0 === 0 // true

Object.is(NaN, NaN); // true
NaN === NaN // false
```

## And remember...

```js
'' == false; // true
Object.is('', false); // false
'' === false; // false
```
