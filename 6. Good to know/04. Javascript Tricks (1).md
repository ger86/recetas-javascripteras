## 4ï¸âƒ£ How many times appears an element in an array?

```js
const countOccurrences = (values, element) => 
    values.reduce((acc, value) => (value === element ? acc + 1 : acc), 0);
countOccurrences([1, 2, 3, 1, 4, 5, 1, 6, 7], 1); // 3
```
