## String conversion

For converting **explicitly** a value to a string you could use the `String()` function:

```js
String(123)  // '123'
```

For converting **implicitly** a value to a string you could use the `+` operator:

```js
123 + ''  // '123'
```

Or backticks:

```js
`${123}`  // '123'
```

## Number conversion
