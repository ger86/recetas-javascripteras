ğŸ’¡ Currying is a transformation that converts a function from being callable as f(a, b, c) to f(a)(b)(c).

ğŸ˜ƒ This transformation allows to get tiny reusable functions.

```
function multiply(a, b) {
    return a * b;
}

function curry(f) {
    return function(a) {
        return function(b) {
            return f(a, b);
        }
    }
}

const curriedMultiply = curry(multiply);
console.log(curriedMultiply(2)(3)); // 6
```
